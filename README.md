# ALMA TRADES (Expo React Native MVP)

ALMA TRADES is a simulated AI-powered trading performance operating system mobile app MVP built with Expo + React Native + TypeScript.

## Stack
- Expo
- React Native
- TypeScript
- React Navigation (Bottom Tabs)

## Run locally
1. Install dependencies:
   - `npm install`
2. Start Expo:
   - `npx expo start`
3. Open with Expo Go on iOS/Android by scanning the QR code.

## Notes
- This is a pure mobile app entry (`App.tsx`) and not a Next.js or web-first project.
- Uses mock data only.
